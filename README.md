# Fake Doctor Prescription Generator

A modern, customizable web app to generate fake doctor prescriptions. Users can input details, select medicines, add notes, customize the look, and print or download the prescription. Hosted on GitHub Pages via GitHub Actions.

## Features
- Input patient and doctor details
- Add multiple medicines
- Add notes
- Customizable theme (colors, font)
- Print or download prescription
- Responsive and modern UI
- Deploys automatically to GitHub Pages

## Setup
1. Clone this repo
2. Run `npm install`
3. Run `npm start` to develop locally
4. Update `homepage` in package.json to your GitHub Pages URL
5. Push to GitHub; GitHub Actions will deploy to Pages

## Customization
- Use the theme customizer to change colors and font
- All fields are editable before generating the prescription

## Deployment
- On push to main, GitHub Actions will build and deploy to GitHub Pages
